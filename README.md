# Desafio Técnico

## Objetivo
Resolver pequenos desafios de programação em C# e responder a questões
conceituais sobre esses desafios.

## Instruções
- O código deve ser armazenado em um repositório público no GitHub e o link
deve ser compartilhado através do e-mail;
- Resolva cada um dos desafios e faça commits frequentes;
- Responda às questões em um arquivo chamado “QUESTIONS.md”.
